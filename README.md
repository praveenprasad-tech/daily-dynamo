# Daily Dynamo ✅

A smart and interactive To-Do List app built with HTML, CSS and JavaScript.

## Features
- Add tasks with a deadline date and time
- Get alarm and popup when deadline is reached
- Tasks marked as Completed or Incomplete automatically
- Filter tasks by All, Active, Completed
- Clean and responsive UI

## Tech Stack
- HTML5
- CSS3
- JavaScript (Vanilla)

## How to Run
1. Clone this repo
2. Open `index.html` in your browser
3. Start adding tasks with deadlines!

## Live Demo
Open `index.html` directly in any browser — no setup needed!
